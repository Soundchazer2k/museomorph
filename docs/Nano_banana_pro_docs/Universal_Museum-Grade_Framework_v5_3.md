### CRITICAL (Generation Failure if Violated)
* **Photorealistic Breach:** If the face looks like a photograph while the body looks like art, FAIL.

* **Anachronism Violation:** If a modern object appears untranslated, FAIL.

* **Edge Failure (v5.3):** Edge construction doesn't match specified type.
* **Likeness Floor Breach (v5.3):** If face drops below likeness floor, IDENTITY LOST.
